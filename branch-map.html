结局与引擎判定（摘要）

  • 圆满·拾光同行：chose_hometown 且好感/信任/自我达阈值 → 完整文案；否则 → 温和版。
  • 圆满·温柔相守：chose_studio，同上。
  • 释然：path_release，高自我且低好感 → 画展释然；否则 → 各自上路。
  • 挽回线：path_redemption（陈默调解后），按好感/信任分「和解之后 / 仍有余悸」。
  • 幕后落幕：陈默调解后转入「车内笔记」分支，触发 ending_observer。
  • 黑暗1/2/3：对应 ending_dark1/2/3 旗帜。
  • 镜像值兜底：mirror 过高且信任偏低时，即使未手动选黑暗线，也会坠入「黑暗2 假面共生」。
  • 遗憾/孤独：ending_regret_pass、ending_lonely。
  • 拉锯：仅 tug_war 且未选家乡/工作室结局时，tug_commit → 双向奔赴，否则 → 拉锯未完。
  • 留白 / 开放式：未命中以上时按数值落到 end_neutral_warm 或 end_neutral_open。
